Random User Directory

Project Overview

Random User Directory is a simple web application that fetches user data from a public REST API and displays it dynamically on the webpage. Users can search and filter the displayed results using the search bar.

Features

- Fetches data from a public REST API using Fetch API
- Displays user information dynamically
- Loading state while data is being fetched
- Error handling for failed API requests
- Search functionality to filter users by name
- Responsive user interface

API Used

Random User API

Endpoint:
https://randomuser.me/api/?results=20

The API provides randomly generated user information including:

- Profile Picture
- Full Name
- Email Address
- Phone Number

Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API

How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run the "index.html" file in a web browser.
